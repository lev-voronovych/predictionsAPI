# Prediction API

Простий API для створення, перегляду та видалення передбачень з підтримкою зображень і типів.

## Локальний запуск

1. Встанови залежності:
   ```bash
   npm install
   ```
2. Створи файл `.env`:
   ```env
   PORT=5004
   MONGODB_URI=твій_рядок_підключення_до_mongodb
   ```
3. Запусти сервер:
   ```bash
   npm start
   ```

Локальна адреса:

http://localhost:5004

---

## Функціонал

- додавання передбачень через форму або API;
- завантаження зображення файлом;
- можливість вказати URL зображення;
- три типи передбачень:
  - Колячний
  - маніглайний
  - від себе

---

## API

### Отримати всі передбачення

GET /predictions

### Відповідь

```json
[
  {
    "_id": "64f1...",
    "title": "Успіх",
    "text": "Сьогодні тобі пощастить.",
    "image": "/uploads/12345.png",
    "type": "Колячний"
  }
]
```

---

### Отримати випадкове передбачення

GET /predictions/random

---

### Отримати передбачення за ID

GET /predictions/:id

---

### Додати передбачення

POST /predictions

#### Форм-data

- `title` — заголовок
- `text` — текст передбачення
- `type` — тип передбачення
- `image` — URL зображення (необов’язково)
- `imageFile` — файл зображення (необов’язково)

#### Приклад

```bash
curl -X POST http://localhost:5004/predictions \
  -F "title=Навчання" \
  -F "text=Ти швидко вивчиш нову технологію" \
  -F "type=Колячний" \
  -F "imageFile=@/path/to/image.png"
```

---

### Оновити передбачення

PUT /predictions/:id

Ті самі поля, що й у POST.

---

### Видалити передбачення

DELETE /predictions/:id

---

## Структура передбачення

```json
{
  "_id": "64f1...",
  "title": "Успіх",
  "text": "Сьогодні тобі пощастить.",
  "image": "/uploads/12345.png",
  "type": "Колячний"
}
```

| Поле  | Тип    | Опис                       |
| ----- | ------ | -------------------------- |
| \_id  | string | Ідентифікатор MongoDB      |
| title | string | Заголовок передбачення     |
| text  | string | Текст передбачення         |
| image | string | Шлях до зображення або URL |
| type  | string | Тип передбачення           |
