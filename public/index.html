📘 Інструкція та API

Як користуватися

  1. Заповни поля у формі додавання.
  2. Натисни кнопку «Додати», щоб створити передбачення.
  3. Кожне передбачення можна видалити кнопкою «Видалити».
  4. Усі дані зберігаються у файлі на сервері.

API маршрути

  • GET /predictions — отримати всі передбачення
  • GET /predictions/random — отримати випадкове
  • GET /predictions/:id — отримати за ID
  • POST /predictions — додати нове
  • PUT /predictions/:id — оновити
  • DELETE /predictions/:id — видалити

Локальна адреса: http://localhost:3067
